ATM Simulator (Client-side Demo)

live website link : https://manishalingala.github.io/Take_home_project_ATM/

Description:
This is an interactive ATM Simulator built using HTML, CSS, and JavaScript. The simulator mimics the basic functionality of an ATM, allowing users to log in with demo accounts, check balances, withdraw and deposit money, change their PIN, and view transaction history.
All operations are performed client-side using your browserâ€™s memory (localStorage). This means no data is sent to a server, making it safe for testing and learning purposes.
The project includes three pre-existing demo accounts to test the system.

Features

1. User login with Account ID and PIN (on-screen keypad or keyboard input)
2. Check account balance
3. Withdraw money (with insufficient funds check)
4. Deposit money
5. Change account PIN
6. Transaction history with timestamps (latest first)
7. Reset all demo accounts to initial values

Open atm_demo.html in your browser.  Enter one of the Account IDs and PINs using the right panel. You can use either the keyboard or the on-screen keypad.Click Login.
After logging in, you can:

View your current balance in dollars        Deposit or withdraw funds    Change your PIN    See your transaction history

Click Logout to exit the session.          Use the Reset button to restore demo accounts to their initial balances and PINs.
